Ionic常用插件使用方法

ionic doc开发帮助文档地址：

https://ionicframework.com/docs/native/transfer/

一、图片选择器imagepicker

git项目地址如下：

https://github.com/Telerik-Verified-Plugins/ImagePicker

插件下载命令如下：

phonegap plugin add https://github.com/Telerik-Verified-Plugins/ImagePicker.git

或者

cordova plugin add https://github.com/Telerik-Verified-Plugins/ImagePicker.git

插件安装命令如下：

cnpm install --save @ionic-native/image-picker

如果网络通顺的话，会很快安装完毕，如果网络原因安装不上，可以选择本地安装。具体方法如下:

1、安装git,这里省略，百度上有

2、git clone https://github.com/Telerik-Verified-Plugins/ImagePicker.git下载到本地

3、找到项目plugins文件，将刚下载的插件copy进去。

4、找到plugins目录下的android.json文件，文件中installed_plugins选项增加条目如下：

        "com.synconset.imagepicker":  {

                        "PACKAGE_NAME":  "io.ionic.starter"

                },

5、找到plugins目录下的fetch.json文件增加条目如下:

        "com.synconset.imagepicker":  {

                "source":  {

                        "type":  "registry",

                        "id":  "https://github.com/Telerik-Verified-Plugins/ImagePicker"

                },

                "is_top_level":  true,

                "variables":  {}

        },

6、工程目录下找到package.json，dependencies中增加条目

    "com.synconset.imagepicker": "2.1.7",

其中版本号"2.1.7"在插件文件夹中的plugin.xml中找
