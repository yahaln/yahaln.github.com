Cordova Ionic环境搭建

一、安装jdk1.8

配置java环境变量如下：

单击“计算机-属性-高级系统设置”，单击“环境变量”。在“系统变量”栏下单击“新建”，创建新的系统环境变量。

  (1)新建->变量名JAVA_HOME，变量值C:\Java\jdk1.8.0_05（即JDK的安装路径）  

  (2)编辑->变量名Path，在原变量值的最后面加上;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

  (3)新建->变量名CLASSPATH,变量值.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

Windows,打开命令提示符，输入javac，出现内容就是配置成功。

二、下载AndroidSDK

  Android  SDK  Platform-tools（平台）

  Android  SDK  Build-tools（编译）

  API23:  SDK  Platform（编译依赖版本）

  Android  Support  Repository（corrswalk需要）

  Google  Repository（corrswalk需要）

  配置安卓环境变量如下:

  设置环境变量全局访问android

  我的电脑  -  右键属性  -  高级系统设置  -  环境变量  

  新建变量名  变量值  

  ANDROID_HOME  D:\ionic\androidsdk(你sdk的路径)然后在path里添加(注意前面的分号)  

  ;%ANDROID_HOME%\platform-tools;%ANDROID_HOME%\tools

  执行android  -h看到命令即安装设置成功

三、Nodejs集成

  Node  Version:  v4.4.7

  Cordova  CLI:  6.2.0

  Ionic  Framework  Version:  1.3.1

  ionic  Version:  1.7.16

安装方法：http://bbs.ionic-china.com/read-7.html

四、安装ionic  cordova

打开cmd（建议管理员运行）将npm映射到淘宝的服务器  

npm  install  -g  cnpm  --registry=https://registry.npm.taobao.org  

然后我们就可以用cnpm命令安装了  

cnpm  install  -g  cordova  ionic（安装  cordova  ionic）

安装完成后运行ionic  -v可以看到版本

五、安装webstorm

网址：https://www.jetbrains.com/webstorm/
